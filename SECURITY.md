# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Peeksy, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email: [security email to be added]

We will respond within 48 hours and work with you to understand and address the issue.

## Scope

Peeksy is a self-hosted application. Security concerns include:
- WebRTC connection security
- Signaling server vulnerabilities
- Room ID predictability
- Password bypass vectors
- WebSocket injection or abuse
